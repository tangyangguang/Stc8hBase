# 测试板建议

## 1. 目标

第一版建议固定一套 `STC8H1K08 TSSOP20` 测试板配置，用来验证基础库所有核心模块。

测试板不是产品板，而是基础库验证板。

目标：

- 覆盖第一版主要 HAL 和驱动。
- 尽量少占引脚。
- 接线清晰，方便示波器测量。
- 所有示例复用同一套 `board_pins.h`。

## 2. 板级目录

建议目录：

```text
board/stc8h1k08_tssop20_demo/
  board_config.h
  board_pins.h
  board_init.c
```

职责：

- `board_config.h`：芯片、时钟、默认波特率、I2C 地址等。
- `board_pins.h`：所有测试外设引脚。
- `board_init.c`：基础引脚模式和板级初始化。

## 3. 默认配置

默认芯片：

```text
STC8H1K08
```

默认封装：

```text
TSSOP20
```

默认系统时钟：

```text
11.0592MHz
```

默认 UART 波特率：

```text
115200
```

保留测试波特率：

```text
9600
```

默认 I2C LCD1602 地址：

```text
0x27
```

备用常见地址：

```text
0x3F
```

## 4. 外设建议

测试板建议包含：

| 外设 | 目的 |
| --- | --- |
| LED | GPIO 输出、状态指示 |
| 按键 | 消抖和事件测试 |
| EC11 | 编码器方向和计数测试 |
| I2C LCD1602 | I2C 和字符显示测试 |
| TM1637 | 二线数码管显示测试 |
| 有源蜂鸣器 | 简单输出控制 |
| 继电器或模拟 LED | 继电器驱动测试 |
| VS1838B/HS0038 | NEC 红外接收测试 |
| 红外发射管 | NEC 红外发射测试 |
| UART 转 USB | 调试输出和串口回显 |

## 5. LCD1602

LCD1602 使用 I2C 转接板。

连接：

```text
VCC
GND
SDA
SCL
```

默认地址先按 `0x27`。

必须提供 I2C 地址扫描示例，用于确认模块实际地址。

## 6. 红外

红外接收：

```text
VS1838B / HS0038
```

红外发射：

- 红外 LED 不建议直接由 MCU 引脚大电流驱动。
- 默认按三极管或 MOS 管驱动设计。
- MCU 输出只作为控制信号。

第一版协议：

```text
NEC
```

## 7. 示波器测点

建议预留或方便夹取以下测点：

- UART TX。
- I2C SCL。
- I2C SDA。
- 红外发射控制信号。
- 红外接收头输出信号。
- PWM 输出。
- 1 个 GPIO 翻转测试点。

没有逻辑分析仪时，示波器是关键时序验证工具。

## 8. 引脚分配原则

具体引脚分配在实现前结合 STC8H1K08 TSSOP20 引脚图确认。

原则：

- 优先保证 UART 调试可用。
- I2C LCD1602 和 TM1637 使用可灵活配置的 GPIO。
- 红外接收优先分配到适合外部中断或定时捕获的引脚。
- 红外发射优先分配到适合 PWM 或定时器输出的引脚。
- 保留至少一个普通 GPIO 用于示波器翻转测试。

## 9. 默认资源分配建议

在不和 STC8H1K08 TSSOP20 实际引脚复用冲突的前提下，测试板优先按以下资源分配：

| 资源 | 默认用途 |
| --- | --- |
| UART | 调试输出和串口回显 |
| Timer0 | 1ms tick |
| Timer2 | 红外接收脉宽计时或红外发射备用载波 |
| PWM | 红外发射 38kHz 载波 |
| 软件 I2C GPIO | I2C LCD1602 |

最终以 STC8H1K08 TSSOP20 引脚图和实测结果为准。

当前核心板第一轮实测接线：

| 外设 | 引脚 |
| --- | --- |
| LED | P1.2，高电平点亮 |
| LCD1602 SDA | P3.2 |
| LCD1602 SCL/CLK | P1.7 |
| EC11 A | P1.0 |
| EC11 B | P1.1 |
| EC11 SW | P5.4 |
| 10K 电位器 | P3.3 |

UART 波特率相关资源在 UART 实现中按芯片手册确定，不在测试板默认分配中预先占用 Timer1。

测试板默认不要求红外发射、红外接收、无源蜂鸣器同时工作。需要测试冲突组合时，单独建立示例并记录资源分配。
